## 当前学习主题

Python Exe 打包

## 技术背景

将 Python 脚本及其依赖（解释器、第三方库）打包成独立的 .exe 可执行文件，
使得没有 Python 环境的 Windows 用户也能直接运行程序。
核心原理是"环境克隆"：把运行时需要的一切塞进一个自包含的包里。

## 工具选型

- **选定工具**：PyInstaller
- **选择原因**：社区最成熟、资料最丰富、支持单文件打包
- **备选方案**：Nuitka（编译为C，保护源码）、cx_Freeze（轻量）、Briefcase（跨平台/移动端）
